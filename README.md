# Color Rush

Blend primary colors to match target shades in a fast-paced puzzle game.

## Description

Tap the colored faucets to dispense drops into your flask. Mix red, blue, and yellow to match the target color shown on screen. Ship completed flasks to earn coins and progress through increasingly challenging levels.

## Controls

### Mobile (Touch)
- **Tap** colored buttons (Red / Blue / Yellow) to dispense drops into the flask
- **Tap** the flask to submit it for matching
- **Tap** targets on the conveyor belt to select them (if multiple orders)
- **Tap** the settings/shop button to access upgrades

### Desktop (Keyboard + Mouse)
- **1** — Dispense Red
- **2** — Dispense Blue
- **3** — Dispense Yellow
- **Click** on colored buttons to dispense drops
- **Click** on the conveyor belt to target flasks
- **F** — Toggle fullscreen

## Features

- 3 primary colors to mix into hundreds of shades
- Daily reward bonuses
- Adaptive difficulty timer
- Shop system with color refills and drop skins
- CrazyGames SDK integration (leaderboard, ads)
